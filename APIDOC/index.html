Poultry Farm API Documentation

Welcome to the Poultry Farm REST API documentation. This API allows you to manage chicken records efficiently.

GET /chickens
Retrieves an array of all chickens currently registered on the farm.

No parameters required.

GET /chickens/:id
Retrieves the details of a single chicken by its unique ID.

Parameters

Name Type Required Description
id Number Required The unique identifier of the chicken (URL path parameter).

Responses

200 OK 
{
  "id": 1,
  "breed": "Rhode Island Red",
  "age": 12,
  "health_status": "Healthy",
  "egg_production_rate": 0.85,
  "created_at": "2023-10-12T08:30:00.000Z"
}
404 Not Found 
{
  "error": "Chicken not found"
}
POST /chickens
Creates a new chicken record with the provided information.

Parameters (Body)

Name Type Required Description
breed String Required The breed of the chicken.
age Number Required The age of the chicken in months.
health_status String Required Current health status of the chicken (e.g. Healthy, Sick).
egg_production_rate Number Required Daily egg production rate as a decimal (e.g. 0.9).

Responses

201 Created 
{
  "message": "Chicken created successfully",
  "id": 3
}
400 Bad Request 
{
  "error": "Validation failed: All fields are required"
}
PUT /chickens/:id
Updates an existing chicken record. You only need to provide the fields you want to update.

Parameters

Name Location Type Required Description
id URL Path Number Required The unique identifier of the chicken to update.
breed Body String Optional The updated breed of the chicken.
age Body Number Optional The updated age of the chicken in months.
health_status Body String Optional The updated health status of the chicken.
egg_production_rate Body Number Optional The updated daily egg production rate.

Responses

200 OK 
{
  "message": "Chicken updated successfully",
  "id": 1
}
400 / 404 Error 
{
  "error": "Validation failed OR Chicken not found"
}
DELETE /chickens/:id
Permanently deletes a chicken record from the farm database.

Parameters

Name Type Required Description
id Number Required The unique identifier of the chicken to delete (URL path parameter).

Responses

200 OK 
{
  "message": "Chicken deleted successfully"
}
404 Not Found 
{
  "error": "Chicken not found"
}
